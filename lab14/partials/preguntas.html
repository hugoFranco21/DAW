Preguntas

¿Qué es ODBC y para qué es útil?

Es una API de estándar abierto para acceder a una base de datos. Es útil porque permite realizar consultas que sean independientes al motor de base de datos, lo que hace más portable el código. [1]

¿Qué es SQL Injection?

Es un tipo de ataque que hace que se ejecuten comandos maliciosos de SQL en una aplicación, logrando obtener información sensible, o incluso afectando la estabilidad de la base al borrar tablas. [2]

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Se pueden hacer varias cosas, como validar el input del usuario, usar queries parametrizadas con prepared statements, no mostrar los errores internos de la base de datos, y mantener un log con los errores para poder solucionarlos en parches subsecuentes. [3]
