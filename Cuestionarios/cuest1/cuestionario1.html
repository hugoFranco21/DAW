Cuestionario 1: BD vs DBMS



  1. ¿Qué es entonces una base de datos - DB?
    Es una colección de datos interrelacionados, que se encuentra almacenada de forma estructurada.[1] Por dato se entiende hecho conocido que puede registrarse y tiene un significado implicito. [2]
  2. ¿En qué casos es conveniente usar bases de datos?
    Cuando la cantidad de datos que se tiene es muy grande, cuando se desea realizar muchas consultas de la información y se quiere eficientizar el proceso, cuando la aplicación para la que se utiliza necesita un control estricto de los datos y las transacciones u operaciones realizadas.
  3. ¿Qué es un sistema de gestión de base de datos - DBMS?
    Es software que ha sido creado para permitir la gestión y el uso eficiente de las bases de datos. Se encarga del acceso, la búsqueda, actualización de los datos y garantizar su integridad. [3]
  4. Enlista y explica las funciones/responsabilidades que tienen los DBMS.
    • Acceso: Poder visualizar la información de la base de datos
    • Búsqueda: Poder definir diferentes filtros para encontrar algún registro o conjunto de registros en particular.
    • Actualización: Realizar las operaciones de inserción y borrado de registros, y que se refleje en la base de datos.
    • Mantenimiento de integridad: Que no se repitan los registros y que todas las transacciones puedan ser verificadas.
    • Elaboración de resúmenes: Se puede generar un resumen de los datos para su análisis, por ejemplo, el total de usuarios que es de cierta ciudad, como se divide la población de acuerdo con sus ingresos, etc.
  5. Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

    El proyecto de Fundamentos de Diseño Interactivo. Nosotros habíamos pensado crear una especie de foro en línea en la que se pudieran preguntar cosas sobre diferentes materias y que alumnos del Tec pudieran contestar. La base de datos hubiera sido utilizada para tener un control de los usuarios que publicaban y contestaban, y para guardar los diferentes hilos y discusiones que se generaran. Como no sabíamos implementar esto abandonamos la idea completamente y terminamos realizando un sitio que si bien ayudaba a los alumnos a encontrar información y enlaces útiles, no guardaba su progreso ni los ayudaba con dudas específicas, lo que reducía su utilidad.
