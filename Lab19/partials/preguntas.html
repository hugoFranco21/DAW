Preguntas

¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications)?

AJAX permite el intercambio de datos con el servidor de manera asíncrona, con lo que aumenta la velocidad de respuesta, llevándola a niveles que sólo se creían posibles con aplicaciones de escritorio. Y a diferencia de otras tecnologías no es necesario instalarla, ya que aprovecha que javascript ya está habilitado en la computadora del cliente. [1]

¿Qué implicaciones de seguridad tiene AJAX?¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

AJAX al ser implementado con JavaScript, necesita que éste se encuentre habilitado en la página, y por la interacción con el servidor, debe validarse para evitar cualquier ataque, ya sea de Cross Site Scripting, Cross Site Request Forgery, o SQL Injection si se utiliza una base de datos. Por lo tanto las validaciones deben ser por ambos lados, cliente y servidor. [2]

¿Qué es JSON?

JSON es el acrónimo de JavaScript Object Notation, y es un formato ligero de intercambio de datos, está diseñado para ser sencillo para las personas de entender y para las máquinas de generarlo. [3]
