¿Qué alternativas a jQuery existen?

jQuery es una librería de JavaScript que permite utilizar menos líneas de código para realizar algunas funciones, esto significa que todo lo que hace jQuery podría hacerse con JavaScript plano. Otras alternativas son cash y UmbrellaJS. [1]
