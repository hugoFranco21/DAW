Preguntas Lab. 4

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    Semejanzas Diferencias
    Son lenguajes Orientados a Objetos [1] Java es un lenguaje de programacion y JavaScript es un lenguaje de scripting [1]
    Pueden ser utilizados para la web Las aplicaciones de Java corren en una maquina virtual, las de JavaScript en un navegador
    Las sentencias se terminan con ; (punto y coma) El codigo de Java debe ser compilado y el de JavaScript es un archivo de texto[2]
    Son multiplataforma La sintaxis de JavaScript es menos estricta que la de Java
  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*) [3]
    • getTime(): Milisegundos desde el 1 de enero de 1970
    • getDay(): Dia de la semana con un numero del 0 al 6
    • getFullYear(): Año actual en formato YYYY (2019, por ejemplo)
    • getHours(): Hora actual del 0 al 23
    • getSeconds(): Segundo actual del 0 al 59
  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*) [4]
    • concat(): Concatenar dos arreglos
    • includes(): Checa si el arreglo tiene cierto elemento
    • indexOf(): Busca un elemento y regresa su indice
    • push(): Agrega un elemento al final del arreglo
    • pop(): Elimina el ultimo elemento del arreglo
  4. ¿Cómo se declara una variable con alcance local dentro de una función?
    Se escribe "var nombreVariable" dentro de las llaves de la funcion, por ejemplo "var num", y se le puede asignar cualquier valor, y este solo sera visible dentro de la funcion. [5]
  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Se puede sobreescribir, perdiendose el valor que tenia previamente contenido. [5]
  6. ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
    El metodo match(). Como parametro se escribe la expresion a buscar y regresa todas las ocurrencias dentro de un array. [6] Se puede usar por ejemplo para checar si una tarjeta de credito es Visa o Master Card con unicamente ingresar los primeros 4 digitos.
