Preguntas

¿Por qué es una buena práctica separar el controlador de la vista?

Para que el usuario no tenga acceso a información privada de otros usuarios, o al funcionamiento interno de la aplicación. También hace que el código se pueda reusar más fácilmente, y más sencillo de implementar cualquier cambio a la interfaz gráfica.[1]

Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?

Los arreglos son los siguientes: [2]

  • $GLOBALS: Recupera las variables globales en cualquier punto del script
  • $_SERVER: Engloba información sobre headers, paths y locaciones de scripts
  • $_REQUEST: Se usa para recoger datos después de subir una forma
  • $_FILES: Variable que puede usarse para subir archivos
  • $_ENV: Se usa para regresar las variables del medio del servidor web [3]
  • $_COOKIE: Recupera las cookies del usuario
  • $_SESSION: Se puede usar para acceder a la información del servidor que esté guardada en la sesión abierta actual

¿Qué es XSS y cómo se puede prevenir?

Es la inyección de scripts maliciosos a sitios confiables. Permite a los atacantes ejecutar scripts en el navegador de otro usuario sin que este pueda saber que es malicioso. Se puede prevenir realizando validaciones a los campos que son ingresados por el usuario y encriptando los datos. [6]
